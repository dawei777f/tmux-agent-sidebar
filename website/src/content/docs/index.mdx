## Features

<CardGrid>
  <Card title="One sidebar, every pane" icon="list-format">
    Tracks Claude Code, Codex, and OpenCode panes across every rmux session and window at
    once, so context switching stops meaning window switching.
  </Card>
  <Card title="Live status, prompts, responses" icon="comment-alt">
    Prompts, tool calls, response previews, background shell state, wait reasons,
    task progress, and subagent trees — all refreshing as the agents work.
  </Card>
  <Card title="Rmux API native" icon="terminal">
    Sidebar multiplexer operations go through rmux Rust APIs, including pane
    input through the SDK broadcast path.
  </Card>
</CardGrid>

## Quick start

Using an rmux plugin manager:

```bash
set -g @plugin 'hiroppy/tmux-agent-sidebar'
```

Reload your rmux config, press `prefix + I` to install, and the install wizard
downloads a pre-built binary or builds from source.

Next, wire up your agent hooks — [Claude Code](/tmux-agent-sidebar/getting-started/claude-code/)
installs as a plugin, [Codex](/tmux-agent-sidebar/getting-started/codex/) is a
one-paste setup inside its own pane, and [OpenCode](/tmux-agent-sidebar/getting-started/opencode/)
uses the bundled plugin bridge.
